# Student Performance Analysis (Data Science)

## Objective
Analyze student marks using statistics, probability and visualization.

## Tools Used
- Python
- NumPy
- Matplotlib

## Concepts Applied
- Mean, Median
- Variance, Standard Deviation
- Histogram & Box Plot
- Probability & Bayes Theorem

## How to Run
1. Install Python
2. Install libraries:
   pip install numpy matplotlib
3. Run:
   python analysis.py
